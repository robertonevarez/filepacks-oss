# CLI

The public v0 CLI surface is intentionally narrow:

```bash
filepacks pack <input> --output <file>
filepacks inspect <file>
filepacks verify <file>
filepacks compare <baseline> <candidate>
```

Commands outside that list are intentionally not part of the stable v0 OSS
surface.

For more CLI context, see `../packages/cli/README.md`.
